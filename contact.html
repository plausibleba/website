Before you write

Quick answers

Installation not working?

The most common issue is running a skill in an existing Cowork task. Always start a new task before typing a slash command. See the full install guide.

What is a ba-skills-bundle.json?

It's the JSON export produced by the /value-stream skill at the end of a session. It contains your capability map, concept model, and value stream in a single schema-validated file. Upload it to the canvas to render your operating model instantly.

Can I use these skills on any domain?

Yes — the skills are domain-agnostic. They work on any business description, from a startup pitch to an enterprise division. See the Dough-To-Door case study for a worked example.

Is this really free?

Yes. The PlausibleBA skills library is free and open source under CC BY-SA 4.0. The Value Cognition Canvas platform has its own pricing for advanced analytical features.

I'm a Guild or IIBA member — can we collaborate?

Absolutely. Reach out to hello@plausibleba.com — we're actively building relationships with the BA and business architecture community.
